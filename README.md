# Innovation & Design Thinking Case Study

## Overview
This project analyzes how Nike, Airbnb, and Oral-B became successful through innovation and design thinking methodologies.

The project explores how these companies applied the 5 stages of design thinking to improve customer experience, solve problems, and create successful products and services.

## Companies Analyzed
- Nike
- Airbnb
- Oral-B

## 5 Stages of Design Thinking
1. Empathize
2. Define
3. Ideate
4. Prototype
5. Test
